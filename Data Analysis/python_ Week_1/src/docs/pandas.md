# Pandas

---

## ¿Qué es Pandas?

Su nombre viene de *panel de datos*. Es una biblioteca de Python de código abierto, fundamental para la ciencia de datos, que proporciona estructuras rápidas y flexibles para manipular, limpiar y analizar datos tabulares (filas y columnas), similar a una hoja de cálculo de Excel.

Se basa en **NumPy** y es ideal para trabajar con:
- Datos estructurados
- Series temporales
- Conjuntos de datos relacionales

---

## Características y usos principales

### Estructuras de datos
Introduce:
- **DataFrames**: tablas bidimensionales
- **Series**: estructuras unidimensionales

---

### Manipulación de datos
Permite:
- Filtrar
- Agrupar
- Ordenar
- Combinar (merge/join)
- Transformar datos complejos con facilidad

---

### Gestión de datos faltantes
Ofrece herramientas potentes para:
- Detectar datos nulos
- Manejar valores perdidos

---

### Lectura y escritura de archivos
Soporta múltiples formatos:
- CSV
- Excel
- SQL
- JSON

---

### Análisis temporal
Especializada en:
- Manejo de series temporales
- Indexación de fechas
- Conversión de frecuencias

---

## Importancia

Pandas es, junto con:
- Matplotlib
- Scikit-Learn

una de las herramientas más utilizadas en el ecosistema de análisis de datos.
